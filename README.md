## About This Project

Vocab2 is an interactive vocabulary learning platform designed to help you expand your language skills efficiently using spaced repetition, active recall, and gamification.

### Key Features
- 📚 Comprehensive word database
- 🎯 Interactive learning modes
- 📊 Progress tracking
- 🌍 Multiple languages
- ⚡ Smart algorithms

## Local Development

This project uses Jekyll and GitHub Pages. To run locally:

```bash
# Install Jekyll (requires Ruby)
gem install jekyll bundler

# Serve locally
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.
